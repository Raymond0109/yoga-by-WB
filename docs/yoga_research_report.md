### 参考资料中发现的缺失体式（流瑜伽常见）

#### 高优先级（流瑜伽核心体式）
1. **上犬式 (Upward-Facing Dog / Urdhva Mukha Svanasana)**
   - 与眼镜蛇式类似但手臂伸直
   - 关键：大腿离地，手臂支撑

2. **半月式 (Half Moon / Ardha Chandrasana)**
   - 单腿站立，身体侧向打开
   - 关键：髋部打开，上方腿伸直

3. **加强侧伸展 (Pyramid / Parsvottanasana)**
   - 双腿前后站立，身体前屈
   - 关键：骨盆正位，脊柱延展

4. **单腿下犬式 (Three-Legged Dog)**
   - 下犬式变体，一腿抬高
   - 关键：髋部保持正位

5. **鸽子式 (Pigeon / Eka Pada Rajakapotasana)**
   - 前腿屈膝，后腿伸直
   - 关键：髋部下沉，脊柱延展

6. **坐立前屈 (Seated Forward Bend)**
   - 坐姿双腿伸直，身体前屈
   - 关键：骨盆前转，脊柱延展

7. **束角式 (Bound Angle / Baddha Konasana)**
   - 坐姿脚掌相对，膝盖打开
   - 关键：脊柱延展，髋部打开

8. **神猴式 (Splits / Hanumanasana)**
   - 竖叉
   - 关键：髋部正位，双腿伸直

9. **反战式 (Reverse Warrior)**
   - 战士二式变体，身体后弯
   - 关键：侧腰延展，上方手臂向后

10. **侧角伸展式 (Extended Side Angle)**
    - 侧角式变体，手臂伸展
    - 关键：侧腰延展，髋部打开

## 三、体式正位标准（基于PDF资料）

### 通用正位原则
1. **脊柱延展**：所有体式都需要脊柱向上延展
2. **核心收紧**：腹部微微收紧，保护腰椎
3. **呼吸配合**：吸气延展，呼气加深
4. **关节对齐**：膝盖与脚尖同向，手腕在肩膀正下方

### 关键体式正位要点

#### 下犬式 (Adho Mukha Svanasana)
- 大小腿前侧收紧上提，后侧延展
- 骨盆向前转动，脊柱延展
- 手掌推地，坐骨向天空方向

#### 战士一式 (Virabhadrasana I)
- 屈膝腿收紧，膝盖与脚尖同向
- 腹部收紧上提，背部延展向上
- 伸直腿髋部尽量向前

#### 战士二式 (Virabhadrasana II)
- 后方腿外旋，大腿收紧上提
- 胸腔打开上提，腹部微微收紧
- 大臂外旋，小臂内旋
- 双手臂向两侧延展

#### 战士三式 (Virabhadrasana III)
- 核心收紧稳定，支撑腿大腿收紧上提
- 脚用力向下踩，上方手臂向上延展
- 上方脚用力向后推

#### 三角式 (Trikonasana)
- 大腿肌肉收紧向上提
- 髋部胸腔向上向外打开
- 上方手臂向上延展有力

#### 侧角式 (Parsvakonasana)
- 伸直腿外旋，大腿收紧向上提
- 屈膝腿收紧尽量靠近支撑手臂
- 上方侧腰延展，手臂尽量向斜上方伸展

#### 幻椅式 (Utkatasana)
- 大腿收紧，膝盖不超过脚尖
- 腹部收紧，脊柱延展
- 双手臂向上延展

#### 下犬式 (Adho Mukha Svanasana)
- 大小腿前侧收紧上提，后侧延展
- 骨盆向前转动，脊柱延展
- 手掌推地，坐骨向天空方向

## 四、流瑜伽常见序列分析

### 流瑜伽 (Vinyasa Flow) 核心序列
1. **拜日式 A (Surya Namaskar A)**
   - 山式 → 手臂上举 → 站立前屈 → 半前屈 → 四柱支撑 → 上犬式 → 下犬式

2. **拜日式 B (Surya Namaskar B)**
   - 山式 → 幻椅式 → 站立前屈 → 半前屈 → 四柱支撑 → 上犬式 → 下犬式 → 战士一式 → 下犬式 → 战士一式 → 下犬式

3. **常见流瑜伽序列**
   - 下犬式 → 三腿犬 → 低弓步 → 战士一式 → 战士二式 → 侧角式 → 反战式 → 三角式 → 下犬式
